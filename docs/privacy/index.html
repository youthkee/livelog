Livelog

プライバシーポリシー

本プライバシーポリシーは、Livelog（以下「本アプリ」）における、ユーザー情報の取り扱いについて説明するものです。

1. 収集する情報について

本アプリでは、以下の情報をユーザーが任意で入力・保存することができます。

  • ライブの開催日時
  • 出演アーティスト
  • ライブ会場
  • バンドメンバー
  • 上演された楽曲のセットリスト
  • その他、ユーザーが記録として入力する情報

これらの情報は、すべてユーザーの端末内にのみ保存されます。

2. 情報の収集・送信について

本アプリは、以下を行いません。

  • 個人情報の収集
  • サーバーへのデータ送信
  • 外部サービスへのデータ共有
  • 広告配信、アクセス解析、トラッキング

保存されたデータが、開発者または第三者に送信されることはありません。

3. アカウント・ログインについて

本アプリは、ログイン機能やアカウント登録を必要とせず、 氏名、メールアドレス、電話番号などの個人を特定できる情報は取得しません。

4. データの保存場所と安全性

本アプリで保存されるデータは、ユーザーの端末内ローカルストレージにのみ保存されます。
外部ネットワークを通じた送信やバックアップは行われません。

5. データの保持期間および削除について

本アプリに保存されたデータは、ユーザーが削除するまで端末内に保持されます。

以下の操作により、データは完全に削除されます。

  • アプリ内の削除機能を使用した場合
  • アプリをアンインストールした場合

6. プライバシーポリシーの変更について

本プライバシーポリシーの内容は、必要に応じて変更されることがあります。
変更後のプライバシーポリシーは、本ページにて公開します。
